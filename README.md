# lethargy_zxspectrum
Исходный код игры Lethargy: The Apathetic Dream, которая вышла в 2003 на ZX Spectrum
(требует 128К для запуска, разрабатывалась на клоне Quorum 1024Mb + TR-DOS)

  Исходники написаны на ассемблере Alasm v4.44, в дополнение к обычным командам ассемблера Z80
добавлены макрокоманды и некоторые другие улучшения
Кодировка текста в образах дискет: IBM866/CP866/DOS 

Для github вытащил исходники из TRD-образов дискет, и перевел их в UTF-8,
для удобства чтения.

Alasm хранит файлы с кодом в страницах памяти, и компилирует код тоже в память, поэтому
на стандартном ZXSpectrum с 128к (и даже с 256к) проект вряд ли получится собрать.

Начинать читать код игры следует с HELL.H
Спрайты игры нарисованы в Spriter и выгружены в виде исходного кода Alasm.
